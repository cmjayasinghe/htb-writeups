# HTB Web Requests Writeups

## Overview
This repository contains my personal writeups, notes, and hands-on exercises for the "Hack The Box Academy – Web Requests** module.

The module focuses on understanding how web applications communicate using HTTP/HTTPS and introduces essential tools and techniques used in web application penetration testing.

This repository is intended for:
- Learning and revision
- Practical command references
- Web penetration testing fundamentals
- API interaction practice
- HTTP request/response analysis

---

# Module Topics Covered

- HTTP Fundamentals
- HTTPS Fundamentals
- HTTP Requests & Responses
- HTTP Headers
- HTTP Methods
  - GET
  - POST
  - PUT
  - DELETE
- Status Codes
- cURL Usage
- Browser DevTools
- API Interaction
- Web Communication Basics
- Web Application Enumeration

---

# Learning Objectives

By completing this module and these writeups, I aim to:

- Understand how web applications communicate
- Analyze and modify HTTP requests
- Work with APIs using cURL
- Use Browser Developer Tools effectively
- Understand request methods and response codes
- Build a strong foundation for Web Application Penetration Testing
